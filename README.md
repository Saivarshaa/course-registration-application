# course-registration-application
A simple full-stack application built using **Java**, **Spring Boot**, and **MySQL** to manage courses and enrolled students. This project demonstrates basic CRUD operations, layered architecture, and API testing with Postman.

## Features

- Enroll students in courses
- View list of available courses
- View list of enrolled students

## Tech Stack

- **Language:** Java , HTML5,CSS3,JavaScript
- **Framework:** Spring Boot
- **Database:** MySQL
- **Tools:** IntelliJ IDEA, Postman, Git, Maven
- **Architecture:** MVC (Controller → Service → Repository)
